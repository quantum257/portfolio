02 ——

Airline Pricing and Revenue Intelligence System

End-to-end analytics project analysing 15,000+ domestic flight records across 10 airlines and 380 routes to identify pricing drivers, route competitiveness, and revenue optimisation opportunities. • Designed a star schema in SQL Server normalising 15,000+ rows into 4 relational tables (fact flights + 3 dimensions), eliminating data redundancy and enabling multi-dimensional analysis • Engineered 10+ derived features in Excel including Source, Total_Stops, Price_Per_Minute, and Duration_mins to enrich raw data for downstream analysis • Wrote 10+ SQL queries across 3 complexity tiers — including window functions (RANK, LAG) and CTEs — to surface pricing anomalies, underpriced routes, and airline competitiveness margins • Built a 3-page Power BI dashboard with custom DAX measures including a Route Competitiveness Score, enabling executive-level pricing strategy recommendations • Identified that IndiGo offers the lowest average fare (₹9,174) while Akasa Air commands the highest premium (₹9,478) — a 3.2% market gap suggesting a highly commoditised pricing environment • Trained a Random Forest regression model in Python achieving R² > 0.80 to predict ticket prices; feature importance analysis confirmed stop count as the strongest pricing driver
